## Common Assignment Patterns (Q4 2025 / Q1 2026)

### RAG Systems (Most Common)
- Document upload (PDF, DOCX, TXT, MD)
- Vector databases (MongoDB Atlas, Qdrant, FAISS, Pinecone, Weaviate)
- LangChain/LangGraph for orchestration
- Citation support
- Question-answering over documents

### Agentic Systems
- ReACT agents
- Tool-calling capabilities
- Multi-step reasoning
- LangGraph for complex workflows
- Autonomous agents with memory

### LLM Integration
- OpenAI API (GPT-4, GPT-4o, Realtime API)
- Google Gemini API
- Anthropic Claude API
- Local LLMs (Ollama, llama.cpp)
- Vector embeddings for semantic search

### GenAI Tasks
- Document Q&A chatbots
- Image generation/editing
- Audio/speech processing
- Conversational interfaces
- Agentic RAG systems

### Tech Stack
| Category | Most Used Tools |
|----------|-----------------|
| **LLM Frameworks** | LangChain, LangGraph, OpenAI, Google Gemini, Anthropic |
| **Vector DBs** | MongoDB Atlas, Qdrant, FAISS, Pinecone, Weaviate |
| **Web Frameworks** | FastAPI, Streamlit |
| **Orchestration** | Docker, Docker Compose |
| **Agent Frameworks** | LangGraph, AutoGen, CrewAI |
