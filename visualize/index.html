What to learn first (no fluff roadmap)

Step 1 - Week 1-2: Python refresh + LLM API basics + structured outputs.
Step 2 - Week 3-4: Build one RAG app end-to-end (ingest, retrieve, answer, eval).
Step 3 - Week 5-6: Add agent tool use with hard limits, retries, and logging.
Step 4 - Week 7-8: Add tests, offline eval, and a basic dashboard for cost/latency.
Step 5 - Week 9-10: Deploy with Docker + one cloud provider and CI/CD.
