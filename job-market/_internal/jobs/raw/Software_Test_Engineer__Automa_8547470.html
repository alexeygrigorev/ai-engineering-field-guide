The Role
Design, build, and maintain automated test frameworks and execute comprehensive UI, API, and database tests. Integrate tests into CI/CD, analyze failures, track defects, drive shift-left practices, and use AI tools to accelerate test generation and failure analysis.
Summary Generated by Built In
Allata is a global consulting and technology services firm with offices in the US, India, and Argentina. We help organizations accelerate growth, drive innovation, and solve complex challenges by combining strategy, design, and advanced technology. Our expertise covers defining business vision, optimizing processes, and creating engaging digital experiences. We architect and modernize secure, scalable solutions using cloud platforms and top engineering practices.

Allata also empowers clients to unlock data value through analytics and visualization and leverages artificial intelligence to automate processes and enhance decision-making. Our agile, cross-functional teams work closely with clients, either integrating with their teams or providing independent guidance—to deliver measurable results and build lasting partnerships.

If you are a smart & passionate team player – then this Software Test Engineer (Automation + AI) opportunity is for you!

We at IMRIEL (An Allata Company) are looking for highly motivated and automation-driven Software Test Engineers with strong expertise in modern test automation practices along with solid manual testing fundamentals. In this role, you will collaborate closely with developers, product owners, QA, and DevOps teams to ensure the delivery of high-quality Web, SaaS, API, and Mobile applications using industry-standard automation tools and CI/CD practices.

This is an excellent opportunity for professionals who are eager to build scalable automation frameworks while strengthening quality engineering capabilities.

Experience: 3 to 5 years.
Location: Vadodara & Pune

What you'll be doing:
• Analyze requirements, user stories, and API contracts to create comprehensive test scenarios and automation strategies
• Build and maintain automated tests using Selenium, Playwright, and Cypress with clean, reusable frameworks (POM, BDD)
• Test APIs thoroughly - authentication, payloads, error handling, and contract validation using Postman and REST automation
• Write SQL queries to validate data integrity, transformations, and business logic in databases
• Run tests across browsers, platforms, and environments (including headless and parallel execution)
• Integrate test suites into CI/CD pipelines and monitor execution quality
• Track defects in JIRA with clear reproduction steps and impact analysis
• Work closely with developers on code reviews and early testing (Shift-Left approach)
• Use AI tools to speed up test generation, analyze failure patterns, review code, and quickly find information in documentation and past test runs
• Maintain test documentation, dashboards, and quality metrics

What you need:
Basic Skills:
• Solid grasp of testing fundamentals - STLC, test design techniques, and best practices
• Hands-on experience testing web apps, SaaS platforms, and APIs (functional, regression, integration, exploratory)
• Proficiency in UI automation with Selenium, Playwright, or Cypress
• Experience building automation frameworks using TestNG and Page Object Model
• API testing skills - REST services, authentication (OAuth/JWT), schema validation
• SQL proficiency - joins, subqueries, aggregations, data validation
• Comfortable with JIRA, Git, and version control workflows
• Programming knowledge in Java, JavaScript, or TypeScript
• Understanding of OOP principles and clean code practices
• Experience working in Agile teams with CI/CD processes

Responsibilities:
• Own the quality of releases by designing and executing comprehensive test strategies across UI, API, and database layers
• Drive automation coverage and reduce regression cycle time through scalable framework design and parallel execution
• Analyze test failures, identify root causes, and collaborate with developers to resolve defects quickly
• Champion Shift-Left practices by participating in requirement reviews, API contract validation, and early test planning
• Maintain CI/CD pipeline stability by monitoring build health, investigating flaky tests, and optimizing execution time
• Contribute to framework improvements including reusable utilities, enhanced reporting, and better test data management
• Participate in Agile ceremonies - provide accurate estimates, identify quality risks, and share testing insights during retrospectives
• Use AI-powered tools and RAG approaches to extract knowledge from technical docs, speed up code reviews, and recognize defect patterns
• Stay current with testing tools and practices, continuously improving test coverage and automation maturity

Good to Have:
• BDD frameworks (Cucumber/Gherkin)
• CI/CD tools (Jenkins, Azure DevOps, GitHub Actions)
• Performance testing (JMeter, K6)
• Cloud platforms (AWS, Azure)
• Cross-browser testing tools (BrowserStack, LambdaTest)
• Docker and containerization basics
• Security testing awareness (OWASP)
• AI coding assistants (GitHub Copilot, Cursor, Claude) for test script development and code optimization
• Vector databases (ChromaDB, Pinecone) for RAG-based test knowledge retrieval
• No-code/low-code testing platforms (Testim.io, Katalon, Functionize) for rapid test authoring
• ISTQB certification

Personal Attributes:
• You think critically and solve problems methodically
• You have an automation-first mindset and strong attention to detail
• You're curious about new tools and willing to learn
• You communicate clearly and collaborate well with teams
• You're proactive, quality-focused, and take ownership of your work

What We Do

Allata (pronounced a-ley-ta) is a strategy, architecture and enterprise-level application development company focused on helping clients enhance or scale business opportunities, create efficiencies and automate processes through custom technologies.
